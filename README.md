# NetfilxClone

There consosts of two parts, including Admin UI and Netfilx Clone UI

## Netfilx Clone UI

### Env and dependencies

- node
- yarn
- vite
- typeScript
- @vue/cli ~3
- axios
- fortawesome
- sass
- vuex
- vue-router
- store

``` shell
    # start project
    yarn dev 

    # build
    yarn build

    # review build projects
    yarn preview 
```

## Admin UI

### Env and dependencies

- node
- yarn
- webpack
- eslint
- @vue/cli ~3
- ant-design-vue
- vuex
- vue-router
- store

``` shell
# Install dependencies
yarn install

# Compiles and hot-reloads for development
yarn run serve

# Compiles and minifies for production
yarn run build

# Lints and fixes files
yarn run lint
```
