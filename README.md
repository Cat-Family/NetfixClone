## Netfilx Clone UI

### Env and dependencies

- node
- yarn
- vite
- typeScript
- @vue/cli ~3
- axios
- fortawesome
- sass
- vuex
- vue-router
- store

``` shell
# Install dependencies
yarn

# start project
yarn dev 

# build
yarn build

# review build projects
yarn preview 
```
